⚠️ Important: Which Sites Can You Scrape?

Before scraping any website, check their Terms of Service. Public sites like OLX, JustDial, and IndiaMART may allow reading public listings but usually prohibit automated bulk scraping. The safest and most legal options are:

✅ Google My Business API — official API, free tier, returns business listings by city
✅ IndiaMART Seller API — official API for business listings
✅ Facebook Marketing API — shows active ads by location
✅ LinkedIn Ads API — for B2B company ads

Using official APIs = legal, reliable, and won't get your IP blocked.
